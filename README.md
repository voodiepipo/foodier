# 🍜 Foodier

Foodier is a restaurant search and mall navigation web application.
## 📸 Screenshots

![Foodier UI](./screenshots/foodier1.png)
## 🚀 Features
- Search restaurants
- View restaurant details
- Admin panel

## 🛠 Tech Stack
- Java (Spring Boot)
- HTML, CSS, JavaScript

## ▶️ How to Run
./mvnw spring-boot:run
